.gitignore y .DS_Store

¿Que es?

.gitignore, es un archivo de texto que le dice a Git que archivos o directorios debe ignorar en el repositorio, es decir, qué archivos no deben ser rastreados, versionados o incluidos en los commits.

¿Como se usa?
  1. Se agrega al documento (touch .gitignore)
  2. Adentrarse al documento
  3. Mencionar lo que se va a ignorar
Cosas a recordar
  • *.log (se puede usar para ignorar todos los archivos con la terminacion .log)
Comandos clave

find . -name .DS_Store -print0 | xargs -0 git rm --ignore-unmatch
